# 혈연(血淵) — MVP 로드맵

> **기준 문서**: `.claude/세계관.md`  
> **MVP 정의**: C1(첫 챕터) 완주 가능한 수준. 핵심 전투 루프 → 마을 귀환 → 보스 정화의 흐름을 플레이어가 경험할 수 있는 빌드.  
> **마지막 업데이트**: 2026-05-11

---

## 진행 현황 요약

| 마일스톤 | 목표 | 상태 |
|----------|------|------|
| M1. 전투 코어 | 세계관의 모든 전투 규칙을 코드로 구현 | 🔄 진행 중 |
| M2. 적 & UI | 모든 적 타입 + 전투 정보 UI 완성 | 🔄 진행 중 |

| M5. 튜토리얼 & 엔딩 | 튜토리얼 씬 + C1 엔딩 최소 1개 | 🔲 미착수 |

## M1 — 전투 코어

**목표**: 세계관 7장(전투 시스템)의 모든 규칙이 코드에 반영된다.

### ✅ 완료

| 항목 | 구현 파일 | 비고 |
|------|-----------|------|
| 플레이어 이동 (이동/점프/대시) | `PlayerMovement.cs` | |
| 이중 자원 (물 HP + 오염도) | `PlayerWaterStats.cs` | |
| Water Tier 0~3 전환 (O키) | `PlayerWaterStats.CycleWaterTier()` | |
| 플레이어 공격 시 HP 소모 | `SkillBase.cs` | Tier별(1/5/15/30) 차등 소모 |
| 기본 공격 (J키, Tier별 범위 차별화) | `BasicAttackSkill.cs` | |
| 광역 베기 (U키) | `WaterStanceMain.cs` | |
| 원거리 공격 (K키, Tier 1~3) | `WaterStanceSub.cs` | |
| 적 AI FSM (Idle/Patrol/Chase/Attack/Hit/Dead) | `EnemyAI.cs` | |
| 적 공격 예비동작 (Windup) | `EnemyAI.cs` | |
| 적 이중 자원 (HP + 오염도) | `EnemyStats.cs` | |
| Sweet Spot 정화/파괴 판정 | `EnemyStats.CheckDeathState()` | |
| 정화된 적 → NPC 전환 | `PurifiedNPC.cs` | |
| 패링 시스템 로직 | `EnemyAttack.cs`, `PlayerController.cs` | |

### 🔲 미완료

| 항목 | 세계관 근거 | 우선순위 |
|------|------------|---------|
| **3단계 지속 HP 소모** | "약 5초간 지속적으로 대량 소모" — 세계관 7-3 | 높음 |
| **패링 시각/청각 피드백** | 애니메이션, 이펙트, 사운드 없음 | 중간 |
| **패링 실패 피해** | 현재 패링 실패 = 일반 피격. 별도 패널티 없음 | 낮음 |

## M2 — 적 & UI

**목표**: 모든 적 타입이 존재하고 전투 정보가 화면에 표시된다.

### ✅ 완료

| 항목 | 구현 파일 |
|------|-----------|
| Player HUD (HP바, 오염도바, Tier 표시, 비네트) | `PlayerHUDBar.cs`, `PlayerTierDisplay.cs`, `PlayerVignette.cs` |
| BossStats (EnemyStats 상속, IsInPurificationRange) | `BossStats.cs` |
| BossUIManager | `BossUIManager.cs` |
| BossRoomTrigger | `BossRoomTrigger.cs` |

### 🔄 진행 중

| 항목 | 현재 Phase | 세계관 근거 |
|------|-----------|------------|
| 적 머리 위 World Space UI (HP + 오염도 + Sweet Spot) | Phase 2 — 미완 | 세계관 7-1, 7-5 |
| 보스 전용 Screen Space UI | Phase 3 — 미완 | 세계관 5-1 |

### 🔲 미완료

| 항목 | 세계관 근거 | 우선순위 |
|------|------------|---------|
| **강(River) 적 타입** | 1~2단계 집행 필요, 정화 후 대장장이/주술사/이야기꾼 — 세계관 5-2 | 높음 |
| **연못 적 차별화** (소형, 0~1단계) | 현재 EnemyStats를 그대로 사용. Tier별 정화 범위 폭 다르게 설정 필요 | 낮음 |

## M4 — 마을 시스템

**목표**: 보스 정화 후 마을로 귀환하고, 중앙 분수에서 HP를 회복할 수 있다.

### 🔲 미완료

| 항목 | 세계관 근거 | 우선순위 |
|------|------------|---------|
| **중앙 분수** (HP 회복, 마을 레벨별 회복량) | "아바타는 분수에서 살점(HP)을 회복할 수 있다" — 세계관 8-2 | ⚠️ 최우선 |
| **강 정화 후 NPC 역할 부여** | `RiverEnemyStats.cs`, `VillageManager.cs` | ✅ 완료 |
| **마을 경제 시스템** | `VillageCurrencyManager.cs` (살점 정수) | ✅ 완료 |
| **NPC 서비스 업그레이드** | 재화 소모 및 단계별 강화 (Level 1~5) | ✅ 완료 |
| **마을 업그레이드** (0→1단계, 분수 소형 활성화) | 세계관 8-3 | 낮음 |

## M5 — 튜토리얼 & 엔딩

**목표**: 신규 플레이어가 핵심 메카닉을 배울 수 있고, C1 클리어 후 엔딩 분기 조건이 동작한다.

### 🔲 미완료

| 항목 | 세계관 근거 | 우선순위 |
|------|------------|---------|
| **튜토리얼 씬** (각성→이동→0단계→패링→정화 집행) | 세계관 9-2 전체 | 높음 |
| **바다의 말투 대화 시스템** | "짧고 단정, 감각적 언어" — 세계관 9-4 | 중간 |
| **엔딩 분기 로직** | 세계관 11-1 규칙 | 중간 |
| — 경로 A: 월드 정화율 0%, 모든 존재 소멸 | 세계관 11-2 | — |
| — 경로 B: 호수 1마리라도 소멸 시 끝없는 전쟁 | 세계관 11-2 | — |
| — 경로 C-1: 모든 호수 정화 후 정지 (≥80%) | 세계관 11-2 | — |
